# Text Analysis Insights

## Overview  
This Jupyter notebook provides tools for comprehensive text analysis. Users can input raw text or extract text from a URL to perform sentiment analysis, summarization, keyword extraction, readability scoring, lexical diversity computation, and text similarity comparison. Visualizations help interpret the results effectively.

## Features  
- Text extraction from URLs  
- Sentiment analysis with polarity and subjectivity  
- Text summarization using Latent Semantic Analysis  
- Keyword extraction via TF-IDF  
- Readability scoring (Flesch Reading Ease)  
- Lexical diversity calculation  
- Text similarity measurement using cosine similarity  
- Interactive plots for sentiment distribution, readability, lexical diversity, word frequency, sentence length, and keyword wordcloud  

## Tech Stack  
- Python  
- Jupyter Notebook / Google Colab  
- Libraries: requests, BeautifulSoup, TextBlob, sumy, scikit-learn, textstat, plotly, wordcloud, matplotlib, numpy, pandas  

## How to Use  
1. Run the notebook in Google Colab or any Jupyter environment.  
2. Choose to enter text manually or provide a URL to extract text automatically.  
3. View the printed analysis results including sentiment, summary, keywords, readability, and lexical diversity.  
4. Optionally, compare similarity with a second text.  
5. Explore interactive visualizations generated for deeper insights.
